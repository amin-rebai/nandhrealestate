## Implementation Tips

1. **Use getText() helper** for all dynamic text
2. **Keep translations consistent** across pages
3. **Test with actual content** from backend
4. **Check RTL layout** for Arabic
5. **Verify form placeholders** are translated
6. **Test language switching** on each page

## Translation Quality Checklist

- [ ] Grammar is correct
- [ ] Terminology is consistent
- [ ] No hardcoded English text remains
- [ ] Placeholders are translated
- [ ] Button labels are translated
- [ ] Error messages are translated
- [ ] Success messages are translated
